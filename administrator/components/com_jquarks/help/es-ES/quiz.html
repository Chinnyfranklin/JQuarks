Test

Los test serán presentados a los usuarios. Son selecciones de series de preguntas. Los test son de dos tipos :

  • Test público
  • Test privado

Test público

Definiendo un test como público, cualquier visitante tendrá acceso para realizar el test.

Test privado

Definiendo un test como privado, sólo estará disponible para los usuarios que elijas de la lista de usuarios registrados. Puedes notifiacar a un usuario la disponibilidad de un test enviándole un e-mail después de relacionarlo con el test.

Configuración del test

Una vez que el test se ha creado puedes modificar su configuración :

Paginar

Puedes elegir paginar tu test o realizarlo en una sola página. Si eleiges el paginar un test debes especificar el número de preguntas a mostrar en cada página. El efecto de deslizamiento está disponible al activarlo de una página a otra.

Publicación

Puedes manejar la disponibilidad de tus test al establecer una fecha de comienzo y fin de publicación
