Test

Os test serán presentados aos usuarios. Son seleccións de series de preguntas. Os test son de dous tipos :

  • Test público
  • Test privado

Test público

Definindo un test como público, calquer visitante terá acceso para realizar o test.

Test privado

Definindo un test como privado, só estará dispoñible para os usuarios que elixas da lista de usuarios rexistrados. Podes notificar a un usuario a dispoñibilidade dun test enviándolle un e-mail despois de relacionalo co test.

Configuración do test

Paxinar

Podes elexir paxinar o teu test ou realizalo nunha soa páxina. Se elixes paxinar un test debes especificar o número de preguntas a mostrar en cada páxina. O efecto de deslizamento está dispoñible ao activalo dunha páxina a outra.

Publicación

Podes manexar a dispoñibilidade dos teus test ao establecer unha data de comenzo e fin de publicación
