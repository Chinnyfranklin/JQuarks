Ensembles de Questions

Les ensembles de questions sont des regroupements de questions. Ils permettent de classer vos questions avant de les affecter à un quiz. Pour ce faire vous disposez de deux types d'ensembles :

  • Ensemble personnalisé
  • Ensemble aléatoire

Ensemble personnalisé

Ce type d'ensemble vous permet de choisir quelles questions vous souhaitez affecter à l'ensemble. Dans un ensemble aléatoire vous pouvez regrouper des questions de catégories homogènes ou hétérogènes. Affecter une question est simple :

  • Utilisez les filtres pour rechercher les questions que vous souhaitez affecter.
  • Ajouter/Enlever une question en un seul clic.

Ensemble aléatoire

Les ensembles aléatoires vous permettent de choisir une catégorie de question à partir de laquelle vous pouvez spécifier un nombre défini de questions à extraire de façon aléatoire.
