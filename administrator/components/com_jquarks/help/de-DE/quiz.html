Quiz

Quize erscheinen im Frontend. Sie bestehen aus einem oder mehreren Fragenkatalogen. Es gibt zwei Arten von Quizen:

  • Öffentliches Quiz
  • Privates Quiz

Öffentliches Quiz

Auf dieses Quiz hat jeder Besucher der Seite Zugriff.

Privates Quiz

Auf diese Art Quiz haben nur Benutzer Zugriff, die vorher aus der Liste der registrierten User ausgewählt wurden. Man kann Benutzer auf neue Quize per Mail hinweisen, nachdem man ihm das Quiz zugewiesen hat.

Konfiguration des Quiz

Die Konfiguration eines erstellten Quiz kann man wie folgt ändern:

Seitenumbruch

Quize können mit allen Fragen auf einer Seite oder mit jeder Frage auf einer einzelnen Seite dargestellt werden. Beim Seitenanbruch kann man festlegen, wieviele Fragen auf einer Seite dargestellt werden. Ein sog. "Slide-Effekt" kan beim Seitenumbruch eingestellt werden, so dass die Seiten animiert weitergleiten.

Veröffentlichen

Man kann die Start- und Endzzeit der Freigabe eines jeden Quiz analog zu den Artikeln in Joomla, einstellen.
