Quiz

Les Quiz sont présentés à vos utilisateurs, ils sont constitués d'ensembles de questions. Les Quiz sont de deux types :

  • Quiz Public
  • Quiz Privé

Quiz Public

En créant un quiz public, tous vos visiteurs y auront accès.

Quiz Privé

Au niveau des quiz privé, le quiz n'est accessible qu'aux utilisateurs que vous affectez depuis la liste des utilisateurs enregistrés sur votre site. Il est possible de notifier un utilisateur de la disponibilité d'un quiz en lui envoyant un mail de notification.

Configuration d'un Quiz

Une fois le quiz créé, il vous est possible de modifier sa configuration :

Pagination

Il est possible de paginer vos quiz ou de les présenter en une seule page. Si vous choisissez de paginer votre quiz vous pourrez définir le nombre de question que vous souhaitez afficher par page. Un effet de "slide" (ou glissement) est également disponible lors du passage d'une page à une autre.

Publication

Il est possible de planifier la publication de vos quiz en définissant une date de début et de fin de publication.
