Respuestas

Una vez que el usuario ha completado tu test, una respuesta será generada y te proporcionará los detalles sobre la puntuación de los usuarios y otra información útil. Es posible mostrar detalles sobre las respuestas de los usuarios, la página de detalles también te permitirá corregir las respuestas en el caso de las respuestas de las preguntas libres.

Detalles de las respuestas

Seleccionando un test desde la lista desplegable tendras acceso a todas las respuestas proporcionadas para aquellos test específicos que posteriormente puedes ver las respuestas de un usuario en concreto y filtrar todas los test sin categoría. Un test no terminado tiene lugar cuando un usuario sólo visualiza el test y no lo responde.

Exportar a CSV

Esta opción te permite exportar el resultado que se muestra actualmente a un archivo .csv permitiendo el posterior análisis de los resultados Si deseas tener todas las respuestas para un test en concreto debes seleccionarlo en el menú desplegable y en la paginación usar la opción de "Desplegar todo".

Borrar Respuestas no terminadas

Esta opción te permite borrar todas las respuestas no terminadas o las respuestas no terminadas de un test o usuario en concreto
