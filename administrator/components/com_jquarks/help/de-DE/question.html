Fragen

Eine Frage zu erstellen, ist einfach und schnell:

  • Einde Frage in den Editor eintragen.
  • Eine Kategorie wählen, in die die Frage passt.
  • Eine oder mehrere Aussagen hinzufügen. (optional)

Aussagen

Die Anzahl der Aussagen ist unbegrenzt. Soviele Aussage wie benötigt hinzufügen. Der Typ der Frage entscheidet über die Auswahlmöglichkeiten für die Aussagen.

Freie Antwort

Wenn keine Aussagen angegeben wurden, erscheint ein Textfeld, in das der Benutzer eine freie Antwort eintragen kann.

Eine Auswahl ist richtig

Wenn nur eine der eingetragenen Aussagen als korrekt angegeben wurde, kann der Benutzer auch nur jeweils eine Aussage auswählen. Die Auswahl erfolg in diesem Falle über sog. Radiobuttons.

Multiple-Choice-Fragen

Wenn mehr als eine Aussage als korrekt markiert wurde, wird die Frage im Multiple-Choice-Verfahren gestellt werden. Die Auswahl erfolgt in diesem Falle über sog. Checkboxes.
