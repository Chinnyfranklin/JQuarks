Fragenkataloge

Ein Fragenkatalog ist eine thematische Gruppierung von Fragen. Er dient dazu, ein Bündel von Fragen einem Quiz zuzuweisen. Es gibt zwei Typen von Fragenkatalogen:

  • Angepasste
  • Zufallsfragen

Angepasste

Dieser Fragenkatalog enthält nur Fragen, die man gezielt ausgewählt hat. Die Reihenfolge wird wie vorgegeben eingehalten. Er enthält Fragen einer oder mehrerer Kategorien. Das Auswählen der Fragen ist einfach:

  • Über die Filterfunktionen die Fragenauflistung durchsuchen
  • Durch einfaches Anklicken die gewünschten Fragen auswählen.

Zufallsfragen

Zufallsfragen werden aus einer Kategroie von Fragen angezeigt. Man kann die Anzahl der gewünschten Fragen angeben, Diese muss aber kleiner sein, als die Anzahl der Fragen in der Kategorie. Aus dem Katalog wird dann die gewünschte Anzahl von Fragen zufällig ausgewählt.
